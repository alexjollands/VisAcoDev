Travelling ants: this scenario shows artificial agents moving around a graph. The aim is to show how the behaviour of ants can be applied to the travelling salesman problem (TSP).

These artificial agents travel from node to node loosely following the rules of the TSP, trying to visit each city once. Laying pheromone as it goes, the red agent favours the closest city (with a shorter connecting path), whilst the blue agent(s) favour the farthest (meaning longer routes).

Turn up the speed and notice that when the red agent travels along a path for the second time it tops up the pheromone already there. The blue agent on the other hand takes so long to get back to the same path that it has almost all evaporated.

This idea, that ants on a shorter route travel around the graph quicker (and therefore lay more pheromone) can be used to create a simple ant colony optimisation (ACO) algorithm to solve complex combinatorial optimisation problems (such as the TSP).
