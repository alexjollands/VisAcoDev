This scenario shows a simple ant colony optimisation (ACO) tackling the travelling salesman problem (TSP).

The artificial ants rapidly traverse the graph going from node to node visiting each exactly once per tour. They lay pheromone as they go, with the stronger red colour indicating higher pheromone levels on that path.

The ants decide which node to visit next using a weighting system. This means they favour shorter paths with high levels of pheromone on them. You can change these weightings yourself and see how it effects the overall outcome.

The green line shows the shortest route the algorithm has discovered so far. With a tiny graph like this the best result found could very well be the actual shortest path, however in almost all real applications it will only ever be a reasonable approximation.

To find a reasonably optimum path in a short amount of time is of course the goal, as definitively finding the best route is too computationally expensive.
